# UART Transmitter in Verilog

This project implements a UART (Universal Asynchronous Receiver Transmitter) transmitter using Verilog HDL.

## Features
- Baud rate generator
- UART transmitter FSM
- 8-bit data transmission
- Start and stop bit framing

## Modules

### baud_generator.v
Generates baud timing ticks from the system clock.

### uart_tx.v
Implements the UART transmitter using a finite state machine.

## Tools Used
- Xilinx Vivado
- Verilog HDL
